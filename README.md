# surfs_up

            SURF'S UP CHALLENGE
            
   Overview of the analysis:
    The purpose of this analysis was to organize and summarize the data collected about the weather in different areas collected to determine where best to set up a surfing business. The goal of analyzing the data was to create the summary statistics based on weather temperatures collected from certain times of the year.
    
    
   Results:
   - The mean of the June temperatures was: 74.944
   - The maximum of the June tempratures was 85.00
   - The mean of the December temperatures was 71.04
